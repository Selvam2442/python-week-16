# 🏥 Medical Insurance Cost Prediction Using Random Forest

## 🚀 Project Overview

The Medical Insurance Cost Prediction System is a Machine Learning project designed to estimate medical insurance charges based on customer demographic and health-related information. Using a Random Forest Regressor, the system predicts insurance costs and identifies the key factors influencing medical expenses.

This project demonstrates how predictive analytics can support insurance providers in premium estimation, risk assessment, and data-driven decision-making.

---

## 🎯 Objectives

- Predict medical insurance charges accurately.
- Analyze factors affecting insurance costs.
- Identify high-impact features influencing premiums.
- Improve insurance pricing strategies.
- Support data-driven decision-making in the insurance industry.

---

## 📂 Dataset

The dataset contains customer information commonly used by insurance providers.

### Features Used

| Feature | Description |
|----------|-------------|
| Age | Customer age |
| Sex | Gender of the customer |
| BMI | Body Mass Index |
| Children | Number of dependent children |
| Smoker | Smoking status |
| Region | Residential region |
| Charges | Medical insurance charges (Target Variable) |
